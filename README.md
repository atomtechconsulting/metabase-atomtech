# AtomTech Metabase

AtomTech's customized distribution of [Metabase](https://github.com/metabase/metabase), the open-source business intelligence platform.

## About

This repository is a fork of [Metabase OSS](https://github.com/metabase/metabase) maintained by [AtomTech Consulting](https://github.com/atomtechconsulting). It includes AtomTech branding customizations on top of the original Metabase codebase.

## Changes from Upstream

- Replaced the default Metabase logo with the AtomTech logo throughout the application UI.
